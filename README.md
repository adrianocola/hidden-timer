# Hidden Timer

A simple web application for players to track their turn time. It's designed to keep the time hidden so it doesn't influence player decisions or the game result (hopefully).

[https://hidden-timer.netlify.app/](https://hidden-timer.netlify.app/)

## Features

- Multi-player support with colors
- Hidden timer to prevent influence on player decisions
- Wake lock to keep the screen on during games
- Responsive layout for mobile and desktop

## How to Use

- **Start Timer:** Tap any player to start their timer.
- **Switch Player:** Tap another player to start their timer and automatically pause the current one.
- **Stop Timer:** Tap the currently active player to stop tracking time.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
